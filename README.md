# Justin Chong Portfolio

Persistent source for Justin Chong's personal portfolio.

## Structure

- `index.html` — production entry point
- `assets/css/site.css` — visual system and responsive layouts
- `assets/js/site.js` — identity grid, navigation, architecture, product, Scatter, and skills interactions
- `assets/images/` — production media
- `planning/` — content strategy, information architecture, and design brief
- `sketches/` — retained visual exploration and reference history

## Run locally

```bash
python3 -m http.server 4175 --bind 127.0.0.1
```

Open <http://127.0.0.1:4175/>.

## Current direction

Concept-led editorial portfolio organized around identity, systems architecture, product portfolio, experience, skills, and résumé. The clinical-AI diagram is a confidentiality-safe conceptual reconstruction, not a disclosure of production security configuration.

## Verification viewports

- Desktop: 1440 × 900
- Tablet: 768 × 1024
- Mobile: 390 × 844

The site must also remain usable with keyboard navigation, reduced motion, and JavaScript-disabled fallback content.
